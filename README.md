# ☕ Advanced Cafe Management System
## Polyglot Database Architecture Project

A full-stack enterprise-level Cafe Management System built using multiple databases, where each database is optimized for a specific task.

This project demonstrates:
- Polyglot Persistence
- Distributed Databases
- Graph Databases
- NoSQL + SQL Integration
- Advanced Database Management Concepts

# 🧠 Polyglot Database Architecture

| Database | Purpose |
|----------|----------|
| Oracle | Core transactions & billing |
| MongoDB | Flexible menu catalog |
| Cassandra | High-speed order logs |
| Neo4j | Recommendation system |
| CouchDB | Reviews & feedback |

---

# 🔄 System Workflow

User places order
↓
Oracle stores order
↓
Cassandra logs analytics
↓
MongoDB retrieves menu data
↓
Neo4j generates recommendations
↓
CouchDB stores feedback
